## Getting Started

To get a local copy up and running follow these simple steps.


### Installation
 
1. Clone the repo
```sh
git clone https://github.com/github_username/repo.git
```
2. Install NPM packages
```sh
npm install
```
3. Install composer
```sh
composer install
```
4. Configure .env
5. Migrate
````sh
php artisan migrate
````
6. Run Dev
```sh
npm run dev 
```
